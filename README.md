# ESP32 Smart Home Light Control & Energy Monitoring System

## Overview

An ESP32-based smart home system that automatically controls lights using a PIR motion sensor and monitors electrical parameters using a PZEM-004T energy meter. The system supports both automatic and manual control modes and displays live data on an LCD.

## Features

* Motion-based automatic light control
* Manual switch override
* Controls 3 relays simultaneously
* Auto shut-off after inactivity
* Real-time monitoring of:

  * Voltage
  * Current
  * Power
  * Energy
  * Frequency
  * Power Factor
* LCD status display
* Serial Monitor diagnostics

## How It Works

* Motion detection turns the lights ON automatically.
* Lights turn OFF after 15 seconds of no motion.
* The manual switch keeps the lights ON regardless of motion.
* Electrical data from the PZEM-004T is displayed on the LCD and Serial Monitor.


## Applications

* Smart Home Automation
* Energy Monitoring
* Smart Lighting Systems
